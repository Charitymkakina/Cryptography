# HASHING
A mathematical algorithm that transforms input data into deterministic, fixed-length representational strings called hash value or digest

# Purpose
Hashing helps verify data integrity, meaning it can show whether the original message has been changed

Hash functions are designed to be one-way, meaning the original input cannot practically be recovered from the `hash`; however, attackers may attempt brute force or dictionary attacks to guess the original data

# Common Hash Algorithms
Legacy- `MD5` `SHA1`
Modern- `SHA224` `SHA256` `SHA384` `SHA512`
Future- `SHA3-224` `SHA3-256` `SHA3-384` `SHA3-512`
